---## 下载

1. 前往 [Navicat | 下载 Navicat Premium 14 天免费 Windows、macOS 和 Linux 的试用版](https://www.navicat.com.cn/download/navicat-premium) 下载链接，根据自己的系统选择合适的安装包。

2. 下载 `Navicat Premium` 注册机：[https://www.aliyundrive.com/s/FFcshNcktyD](https://www.aliyundrive.com/s/FFcshNcktyD)  

## 安装

无需特殊配置，一路下一步即可。

## 注册

::: tip
注册之前请先退出杀毒软件以及 Navicat Premium 进程
:::

### Patch

1. 选择好要激活的版本之后点击右侧的 `Patch` 键

![file](https://cdn.vuepress.learnku.fit/Navicat-Premium%E7%9A%84%E5%AE%89%E8%A3%85%E4%B8%8E%E6%B3%A8%E5%86%8C/patch.jpg)

2. 选择之前安装好的 `Navicat Premium` 的 `exe` 文件

![file](https://cdn.vuepress.learnku.fit/Navicat-Premium%E7%9A%84%E5%AE%89%E8%A3%85%E4%B8%8E%E6%B3%A8%E5%86%8C/select_exe.jpg)

3. 出现如下 `Info` 则代表第一步成功

![file](https://cdn.vuepress.learnku.fit/Navicat-Premium%E7%9A%84%E5%AE%89%E8%A3%85%E4%B8%8E%E6%B3%A8%E5%86%8C/patch_success.png)

### License. Product and Language

如下图所示，保持默认选择即可

![file](https://cdn.vuepress.learnku.fit/Navicat-Premium%E7%9A%84%E5%AE%89%E8%A3%85%E4%B8%8E%E6%B3%A8%E5%86%8C/license_product_language.png)

### Resale License

如下图所示，保持默认选择即可

![file](https://cdn.vuepress.learnku.fit/Navicat-Premium%E7%9A%84%E5%AE%89%E8%A3%85%E4%B8%8E%E6%B3%A8%E5%86%8C/resale_license.png)

### Keygen / Offline Activation

1. 点击右侧的 `Generate` 键，就会在左侧生成一个许可证密钥，复制许可证密钥后打开 `Navicat Premium`

![file](https://cdn.vuepress.learnku.fit/Navicat-Premium%E7%9A%84%E5%AE%89%E8%A3%85%E4%B8%8E%E6%B3%A8%E5%86%8C/generate.png)

2. 打开 `Navicat Premium` 后点击注册

::: tip
点击注册前先断开网络连接，并且关闭网络代理  
大家不用纠结图中的 Request Code 和许可证密钥对应不上，楼主因为网络代理的原因生成了多个许可证密钥，所以导致的截图中的 Request Code 和许可证密钥对应不上
:::

![file](https://cdn.vuepress.learnku.fit/Navicat-Premium%E7%9A%84%E5%AE%89%E8%A3%85%E4%B8%8E%E6%B3%A8%E5%86%8C/open.png)

3. 粘贴注册机生成的许可证密钥，然后点击激活

![file](https://cdn.vuepress.learnku.fit/Navicat-Premium%E7%9A%84%E5%AE%89%E8%A3%85%E4%B8%8E%E6%B3%A8%E5%86%8C/activation.png)

4. 点击激活后会提示因为激活服务器暂时不可用，所以你的许可证未能激活，这里我们选择手动激活

![file](https://cdn.vuepress.learnku.fit/Navicat-Premium%E7%9A%84%E5%AE%89%E8%A3%85%E4%B8%8E%E6%B3%A8%E5%86%8C/manual_actiation.png)

5. 点击手动激活后会生成一个请求码

![file](https://cdn.vuepress.learnku.fit/Navicat-Premium%E7%9A%84%E5%AE%89%E8%A3%85%E4%B8%8E%E6%B3%A8%E5%86%8C/request_code.png)

6. 将请求码复制到注册机中的 `Request Code` 里面，  
   之后点击 `Activation Code` 下面的 `Generate` 按钮就会生成一个激活码，  
   将激活码复制到 `Navicat Premium` 中的激活码框里，点击激活即可完成激活

![file](https://cdn.vuepress.learnku.fit/Navicat-Premium%E7%9A%84%E5%AE%89%E8%A3%85%E4%B8%8E%E6%B3%A8%E5%86%8C/use_request_code_get_activation_code.png)

7. 出现如下弹框则代表激活成功

![file](https://cdn.vuepress.learnku.fit/Navicat-Premium%E7%9A%84%E5%AE%89%E8%A3%85%E4%B8%8E%E6%B3%A8%E5%86%8C/activation_success.png)

8. 点击确定之后即可打开 `Navicat Premium`

![file](https://cdn.vuepress.learnku.fit/Navicat-Premium%E7%9A%84%E5%AE%89%E8%A3%85%E4%B8%8E%E6%B3%A8%E5%86%8C/finish.png)
